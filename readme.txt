== Description ==

Integrates PayPal payments into BA Book Everything plugin.

= Features =
*   PayPal payments

== Installation ==

1. Upload the 'payment-gateway-for-ba-paypal' folder to the '/wp-content/plugins/' directory
1. Activate the "Payment Gateway for BA via PayPal" plugin through the 'Plugins' menu in WordPress
1. Go to "BA Settings" menu, click "Payments" tab and make main settings
